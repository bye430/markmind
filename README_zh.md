# MarkMind — Markdown 脑图可视化工具

将 Markdown 文档实时渲染为 XMind 风格的交互式思维导图，支持画布直接编辑、LaTeX 公式、富文本、多种主题与布局，并具备自适应增量更新与会话缓存。

## 快速开始

无需安装，用浏览器打开即可使用：

```bash
# 方式一：直接打开
open index.html   # macOS
xdg-open index.html   # Linux

# 方式二：本地服务器（推荐，避免部分 API 限制）
python3 -m http.server 8765
# 访问 http://localhost:8765
```

## 功能概览

### Markdown ↔ 脑图双向同步

- 左侧编辑器输入 Markdown，右侧实时生成脑图；在画布上编辑节点，左侧 Markdown 同步更新。
- 编辑器光标与画布选中节点双向联动：点击画布节点会滚动并高亮对应行；在编辑器中移动光标，画布会平滑聚焦到对应节点。

**支持的 Markdown 语法：**

| 语法 | 说明 |
|------|------|
| `#` ~ `######` 标题 | 对应树形节点层级（根 / 一级 / 二级） |
| `-` / `*` / `+` 列表项 | 叶子节点，支持缩进嵌套 |
| `**粗体**` `*斜体*` `` `代码` `` | 节点内富文本渲染 |
| `$...$` / `$$...$$` 等 | LaTeX 公式（见下表） |

### LaTeX 公式渲染

节点内的 LaTeX 由 [KaTeX](https://katex.org/) 渲染：

| 语法 | 类型 | 示例 |
|------|------|------|
| `$...$` | 行内 | `$E = mc^2$` |
| `\(...\)` | 行内 | `\(\alpha + \beta\)` |
| `$$...$$` | 独立块 | `$$\int_0^\infty e^{-x^2} dx$$` |
| `\[...\]` | 独立块 | `\[\sum_{n=1}^{\infty} \frac{1}{n^2}\]` |

### 画布交互式编辑

在脑图画布上直接操作节点，所有操作会同步回 Markdown：

| 操作 | 方式 | 说明 |
|------|------|------|
| 选中节点 | 单击 | 高亮节点，编辑器滚动到对应行 |
| 多选 | `Ctrl` + 单击 | 追加/取消选中 |
| 框选 | 空白区域拖拽 | 矩形框内节点全部选中 |
| 编辑文字 | 双击节点 / `F2` | 内联输入框，Enter 确认、Escape 取消 |
| 新建兄弟节点 | 选中后 `Enter` | 同层级下方插入新节点 |
| 新建子节点 | 选中后 `Tab` | 当前节点下插入子节点 |
| 删除节点 | `Delete` / `Backspace` | 删除选中节点（支持多选批量删除） |
| 折叠/展开 | 点击节点旁 `+`/`−` | 折叠或展开该节点子树 |
| 拖拽移动 | 按住节点拖拽 | 更换父节点或调整兄弟顺序，有连接线预览 |

**拖拽规则：** 只能挂到高一级的标题下（如二级→一级）；拖到兄弟附近可调整排序；画布会禁用原生拖拽与框选时的文本选择，避免与浏览器默认行为冲突。

### 撤销与恢复

- **Ctrl+Z** 撤销、**Ctrl+Y**（或 **Ctrl+Shift+Z**）恢复。
- 独立于编辑器文本的撤销栈，仅针对画布上的结构操作：创建、删除、移动、重命名、排序。

### 布局与主题

**布局方向（工具栏下拉）：**

- 向右展开（默认）
- 左右分布（Mind Map 风格）
- 向下展开（组织架构图）

**主题（工具栏下拉）：** 默认蓝 / 海洋蓝 / 森林绿 / 日落橙 / 黑白极简

### 画布视图

| 操作 | 方式 |
|------|------|
| 平移 | `Alt` + 左键拖拽，或中键拖拽；或在「框选/拖拽」切换为拖拽模式后直接左键拖拽 |
| 缩放 | 鼠标滚轮（以光标为中心） |
| 适应画布 | 工具栏「⊞ 适应」/ **Ctrl+0** |
| 居中根节点 | 工具栏「⌖ 居中」 |
| 画布模式 | 工具栏「▣ 框选」⇄「✥ 拖拽」：框选模式可拖出矩形多选；拖拽模式左键直接平移画布 |
| 小地图 | 右下角缩略图，点击拖拽可跳转视图；点击 **−** 收起为小按钮，再点 **▤** 展开 |
| 触屏 | 双指捏合缩放、单指平移 |

### 菜单与文件

右上角 **☰ 菜单** 提供：

- **文件**：打开 (Ctrl+O)、保存 (Ctrl+S)、另存为
- **编辑**：撤销 (Ctrl+Z)、恢复 (Ctrl+Y)
- **清空缓存**：下次打开恢复为默认初始界面
- **导出 SVG**：导出当前脑图为矢量图

**◐ 编辑** 按钮（或 **Ctrl+E**）：显示/隐藏左侧 Markdown 编辑器面板。

| 操作 | 快捷键 |
|------|--------|
| 打开 .md | **Ctrl+O** / 菜单「打开」/ 拖拽文件到窗口 |
| 保存 | **Ctrl+S** / 菜单「保存」 |
| 另存为 | 菜单「文件 → 另存为」 |
| 切换编辑器 | **Ctrl+E** / 工具栏「◐」 |
| 适应画布 | **Ctrl+0** |

### 会话缓存

- 编辑内容、布局、主题、小地图/编辑器面板状态、视图变换等会自动写入浏览器本地缓存。
- 下次打开页面时自动恢复上次的编辑状态。
- 菜单中「清空缓存」会清除缓存并提示；下次打开将显示默认 Markdown 内容。

### 未保存提示

有未保存修改时关闭或离开页面，浏览器会弹出确认提示。

## 项目结构

```
├── index.html          # 页面入口
├── style.css           # 样式（主题、动画、菜单、节点）
├── parser.js           # Markdown → 树结构解析，LaTeX/富文本检测
├── layout.js           # 树形布局（Reingold-Tilford 变体），节点尺寸测量
├── renderer.js         # SVG 增量渲染，foreignObject 内嵌公式与富文本
├── interactions.js     # 缩放、平移、小地图、画布模式、触屏
├── canvas-editor.js    # 画布编辑状态机（选中/编辑/框选/拖拽）
├── undo-manager.js     # 结构操作撤销/恢复栈
└── app.js              # 应用入口，双向同步与会话缓存
```

## 技术要点

- **增量更新**：通过 `_key` 与节点 ID 匹配与状态转移，仅更新变化节点，其余节点平滑动画到新位置。
- **布局**：改进的 Reingold-Tilford 算法，支持可变节点宽高与三种展开方向。
- **公式与富文本**：LaTeX 与加粗/斜体/代码在 SVG 内通过 `foreignObject` + KaTeX/HTML 渲染；布局阶段用隐藏 DOM 预测量尺寸；长文本自动换行且不在公式/富文本中间断行。
- **画布状态机**：idle / selected / editing / marquee / dragging 等状态，所有画布操作序列化回 Markdown。
- **依赖**：仅通过 CDN 引入 KaTeX，其余为原生 JS/CSS/SVG。

## 浏览器兼容

支持现代浏览器（Chrome / Firefox / Safari / Edge）。Chrome 下可选用 File System Access API 实现更完整的“另存为”体验。
