# MarkMind

MarkMind 是一款在浏览器中运行的轻量工具：左侧 Markdown 编辑，右侧脑图实时呈现。两侧可同步编辑，便于在文字大纲与图形结构之间自然切换。

## 使用方式

用现代浏览器直接打开 `index.html`，或对项目目录启用任意静态文件服务（例如 `python3 -m http.server`），再访问终端中提示的地址即可。

## 功能概览

- **双端编辑** — 支持在编辑器与画布上直接修改内容。
- **外观与布局** — 通过工具栏调整主题与版式。
- **本地会话** — 数据保存在本机浏览器中，直至在应用菜单中主动清除。
