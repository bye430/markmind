# MarkMind

MarkMind is a lightweight, browser-based tool that pairs a Markdown editor with a live mind map. Edits on either side stay in sync, so you can outline in text and refine structure visually without switching contexts.

## Getting started

Open `index.html` in a modern browser, or serve the project folder with any static file server (for example `python3 -m http.server`) and open the URL shown in the terminal.

## Features

- **Dual editing** — Work in the text editor or directly on the canvas.
- **Appearance** — Adjust themes and layout from the toolbar.
- **Local persistence** — Your session is kept in the browser until you clear it from the app menu.
